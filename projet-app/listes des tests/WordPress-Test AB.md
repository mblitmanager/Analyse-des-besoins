**Test de niveau WordPress**

**Initial**

**1) Quels sont les 2 éléments indispensables pour avoir un site internet ?**

1.  Un ordinateur puissant et une connexion fibre optique.
2.  Un compte Facebook professionnel et une carte de visite avec un QR Code.
3.  Un nom de domaine (l'adresse du site) et un hébergement (l'espace de stockage sur un serveur).✅
4.  Je ne sais pas

**2) Où puis-je télécharger le logiciel WordPress ?**

1.  Sur site officiel : https://fr.wordpress.org/✅
2.  Dans une boutique informatique
3.  sur Google Play Store ou App Store
4.  Je ne sais pas

**3) Pour vous, qu'est-ce que WordPress ?**

1.  Un logiciel de traitement de texte comme Word.
2.  Un outil (CMS) qui permet de créer et gérer un site web sans forcément coder. ✅
3.  Un hébergeur de site web uniquement.
4.  Je ne sais pas

**4) À quoi sert principalement un thème dans WordPress ?**

1.  À définir l'apparence visuelle, la mise en page et le design du site.✅
2.  À protéger le site contre les attaques de pirates et les malwares.
3.  À stocker la base de données de tous les articles et commentaires du site.
4.  Je ne sais pas

Passage au niveau basique si réponse 4/4

**Basique**

1**) A quoi sert une extension WordPress ?**

1.  Ajouter des fonctionnalités à WordPress.✅
2.  Changer l'apparence graphique du site.
3.  Ajouter des utilisateurs.
4.  Je ne sais pas

**2) Si vous souhaitez changer votre mot de passe ou votre adresse e-mail de profil, où allez-vous ?**

1.  Dans le fichier wp-config.php.
2.  Dans la gestion de votre compte via le tableau de bord de WordPress.✅
3.  Dans le menu "Thèmes".
4.  Je ne sais pas

**3) Quelle est la différence entre un "Article" et une "Page" ?**

1.  Il n'y en a pas, c'est la même chose.
2.  L'Article est payant, la Page est gratuite.
3.  La Page est pour du contenu dit statique (Contact, À propos), l'Article est pour du contenu chronologique (Actualités).✅
4.  Je ne sais pas

**4) Vous souhaitez ajouter une fonctionnalité de formulaire de contact. Où allez-vous ?**

1.  Dans l'onglet "Apparence".
2.  Dans l'onglet Extensions" (Plugins).✅
3.  Dans l'onglet "Réglages" > "Discussion".
4.  Je ne sais pas

**5) Je souhaite modifier les couleurs de mon site WordPress, que dois-je faire ?**

1.  Modifier les réglages du site.
2.  Coder le HTML.
3.  Personnaliser le thème.✅
4.  Je ne sais pas

**Opérationnel**

**1) À quoi servent les "Permaliens" dans les réglages de WordPress ?**

1.  À changer la langue du site
2.  À définir la structure des adresses URL des pages✅
3.  À sauvegarder le site sur un disque dur
4.  Je ne sais pas

**2) Qu'est-ce qu'un "Menu" dans WordPress ?**

1.  La liste des ingrédients pour faire fonctionner le site.
2.  L'élément de navigation qui permet aux visiteurs d'accéder aux différentes pages. ✅
3.  Un outil pour changer les couleurs du site.
4.  Je ne sais pas

**3) Dans WordPress, quelle est la fonction principale d'un "Article" ?**

1.  Créer du contenu permanent et fixe, comme une page "Contact"
2.  Publier des contenus actualisés qui s'affichent par ordre chronologique, du plus récent au plus ancien. ✅
3.  Modifier uniquement le design des couleurs et la police d'écriture de tout le site.
4.  Je ne sais pas

**4) Dans quel répertoire sont situées les extensions (plugins) de WordPress ?**

1.  wp-content/uploads
2.  wp-include/wp-plugins ✅
3.  wp-content/plugins
4.  Je ne sais pas

**5) Sous quelle forme l'URL d'un article est-elle la plus optimisée pour le SEO ?**

1.  www.mon-site.com/?p=64631
2.  www.mon-site.com/exemple-article-64631
3.  [www.mon-site.com/exemple-article](http://www.mon-site.com/exemple-article) ✅
4.  Je ne sais pas
